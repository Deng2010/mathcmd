# mathcmd
![](./mathcmd.png)\
A simple program to solve math problems.

## install

### install from source code
```bash
git clone https://github.com/Deng2010/mathcmd
cd mathcmd
mkdir build
cd build
cmake ..
make
sudo make install
```

### install from package manager

Arch

You can get the .pkg.tar.zst file from the RELEASE page
